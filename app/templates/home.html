How Keys Are Used

Brave Search API key

The app uses Brave only during the discovery step when you enter a software name without a direct documentation URL.

It does not use Brave for crawling pages, exporting Markdown, or downloading final artifacts.

To stay within the free tier, discovery uses a small query budget, prefers official docs style searches, caches repeated lookups, and degrades gracefully on rate limits.

Optional LLM Assist

OpenAI API key

The app uses OpenAI only when deterministic inspection is ambiguous. The current MVP uses it as an optional fallback for uncertain source-type classification.

OpenAI is not required for crawling, HTML extraction, Markdown conversion, manifest generation, or ZIP export.

If the key is missing or the client cannot initialize, the app continues in deterministic mode.
